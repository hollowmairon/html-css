Desafio das Tags

Agora é a hora de mostrar o que você esta aprendendo em HTML5. Esse exercício faz uma lista de coisas que podem ser adicionadas em uma página. O desafio aqui é arrumar uma maneira que, ao passar o mouse sobre um termo, mostre qual é a tag utilizada para fazer isso.

  • Parágrafos
  • Títulos
  • Imagens
  • Abreviações
  • Códigos-Fonte
  • Listas numeradas
  • Listas com demarcadores
